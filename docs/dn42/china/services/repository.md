# Software Repository

Strategic Explorations hosts a mirror for select open-source projects to provide faster access within the DN42 network.

## eweOS Repository

!!! success "This is enlisted as one of the [eweOS official mirrors](https://os.ewe.moe/download)."

### About

[eweOS](https://os.ewe.moe) is an musl-based, lightweight, general-purpose Linux distribution, which adopts musl libc and busybox to the latest versions of software with a rolling-release model.

### Link

`https://mirrors.nia.dn42/eweos`

### Configuration Guide

To use this mirror on your eweOS system, edit your `/etc/pacman.d/mirrorlist` and enable/add `Server = https://mirrors.nia.dn42/eweos/$repo/os/$arch`.
