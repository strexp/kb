# IANA Accessible Services

These services are accessible from the public Internet (IANA) to facilitate DN42 network debugging and information retrieval.

## DN42 Registry Viewer

A web-based 3D interface to browse the DN42 git registry (Whois data). WebGL is required.

`https://registry42.strexp.net`

## DN42 Map Viewer

3D visualization of the DN42 network topology based on current BGP table dumps. WebGL is required.

`https://map42.strexp.net`

## DN42 ISP Viewer

`https://isp42.strexp.net`

## DN42 Registry Timeline Viewer

`https://timeline42.strexp.net`
